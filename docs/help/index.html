+ Do I need to install anything?
No! Snipt is portable and requires zero installation. Download the file and run it immediately. No setup, no registration needed.
+ How do I start using Snipt?
Simply download Snipt, run the executable, and use the keyboard shortcut to start capturing screenshots. It's that easy!
+ What are the system requirements?
Snipt works on Windows 7 and above. Minimal storage space required (less than 100MB). Works on any modern PC.

+ Can I annotate my screenshots?
Yes! Snipt includes built-in annotation tools. Add arrows, text boxes, highlights, and shapes to clearly mark important areas.
+ What capture modes are available?
Full screen capture and area selection. Choose to capture your entire screen or select a specific region to focus on important details.
+ Is there a screenshot history/gallery?
Yes! All your screenshots are automatically saved to your Snipt gallery. Access them anytime to reference or reuse.

üîí Sharing & Security

+ How do I share screenshots?
After capturing, you can copy the screenshot to clipboard or upload it to get a secure shareable link instantly. Share links via email, Slack, or any messaging platform.
+ Are my screenshots encrypted?
Yes! Shareable links are encrypted. Your screenshots are stored securely in the cloud. Only people with the link can access them.
+ Can I set expiration on shared links?
Yes! You can set links to expire after a specific time or limit them to a single view for extra security.
+ Is my data private?
Absolutely. We use Supabase for secure cloud storage with end-to-end encryption. Your screenshots are never shared or used for any other purpose.

üõ†Ô∏è Troubleshooting

+ The app won't start/crashes
Try running as Administrator. If the issue persists, ensure Windows 7+ and .NET Framework 4.6+ are installed. Check our GitHub issues for solutions.
+ Screenshots won't upload to cloud
Check your internet connection. Ensure you're logged in to Snipt. Try uploading again. Contact support if the issue continues.
+ The hotkey isn't working
Check if another app is using the same hotkey. Customize the hotkey in Snipt settings. Ensure Snipt is running before using the hotkey.

üí° Best Practices

+ How can I take better screenshots?
Focus on clarity - capture only what's essential. Use annotations to highlight key areas. Avoid capturing sensitive info. Use consistent zoom levels.
+ How should I organize my screenshots?
Use descriptive names for screenshots. Tag them by project or category. Create folders for different use cases (bugs, design, documentation, etc.).
+ Tips for team collaboration
Share links in your team chat or Slack. Use annotations for clear feedback. Keep screenshots organized so team members can find them. Use expiring links for sensitive info.
