# Snipt Landing Page

## Configuration

For Supabase authentication to work, update `auth.js` with your credentials:

```javascript
const SUPABASE_URL = 'your-project-url.supabase.co';
const SUPABASE_ANON_KEY = 'your-anon-key';
const DASHBOARD_URL = 'https://dashboard.usesnipt.com'; // or http://localhost:3000 for development
```

Get these values from your Supabase project settings.
