# Snipt - Professional Screenshot Tool

## Overview
Snipt is a lightweight and professional screenshot application for Windows. It provides users with quick and easy ways to capture their screen, select specific areas, and save or share screenshots.

## Features
- ✅ Full screen screenshot capture
- ✅ Area selection with visual overlay
- ✅ Save screenshots to Downloads folder with timestamp
- ✅ Copy screenshots to system clipboard
- ✅ Professional UI with gradient background
- ⏳ Upload and share link in Cloud service (Phase 3)

## Performance Goals
- Executable size: < 10MB
- Startup time: < 500ms
- Memory usage: < 50MB at idle
- Instant area selection UI
